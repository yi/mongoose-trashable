# mongoose-trashable

a mongoose plugin adding ablity to model instance to be trashed (soft deleted) / untrashed

## Install
Install the module with:

```bash
npm install mongoose-trashable
```

## Usage
```javascript
var mongoose_trashable = require('mongoose-trashable');
```
